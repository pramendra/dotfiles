# Browser Profile Aliases

Quick reference for browser profile switching aliases in `.zshrc`.

## Arc Browser

| Alias | Profile | Use Case |
|-------|---------|----------|
| `arc-personal` | Personal | Personal browsing |
| `arc-sg` | SG | SG work |
| `arc-freee` | freee | freee work |
| `arc-globis` | Globis | Globis work |
| `arc-dotpkg` | dotPKG | dotPKG work |
| `arc-power` | Profile 1 | Power user setup |
| `arc-media` | Profile 2 | Media consumption |
| `arc-bare` | Profile 3 | Minimal setup |
| `arc` | Default | Default profile |

## Usage

```bash
# Open Arc with SG profile
arc-sg

# Open Dia with minimal bare profile
dia-bare

# Open Chrome with freee work profile
chrome-freee
```

## Activation

```bash
reload  # or: source ~/.zshrc
```
